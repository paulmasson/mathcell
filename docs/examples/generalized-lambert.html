A generalized Lambert function as described below:

The exponential function can be generalized by the expression

\[ f(x) = \left[ 1 + (1-n) x \right]^\frac{1}{1-n} \]

The derivative of this function is a power of itself

\[ \frac{d f(x)}{dx} = f(x)^n \]

and matches the exponential function exactly in the limit \( n \to 1 \). This prompts one to consider using it to define a Lambert-type function:

\[ W(x) \left[ 1 + (1-n) W(x) \right]^\frac{1}{1-n} = x \]

The principle branch of this function is evaluated by numerical root finding.
