Gyroid minimal surface generated using marching cubes:

The surface is defined implicitly by

\[ \sin x \cos y + \sin y \cos z + \sin z \cos x = 0 \]

The generated surface is restricted to the radial range determined by the slider input. The coloration goes from red at the origin to red at the outer extent, scaled by the ratio of radial variable to its maximum.

This example illustrates some of the inner workings of MathCell. These features may be included in the base code in future releases.
